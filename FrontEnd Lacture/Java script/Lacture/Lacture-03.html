what is operator

In computer science, an operator is a character or characters that determine the action that is to be performed or considered. There are three types of operator that programmers use: arithmetic operators, relational operators, and logical operators.

types of operator

Here are some examples of how to use operators in JavaScript:
  • Arithmetic operators: +, -, *, /, %, ++, --
  • Assignment operators: =, +=, -=, *=, /=, %=, **=, &=, |=, ^=, <<=, >>=, >>>=
  • Comparison operators: ==, ===, !=, !==, <, >, <=, >=
  • Logical operators: &&, ||, !
  • Conditional (or ternary) operators: ?:
  • String operators: +, +=, +=
  • Type operators: typeof, instanceof
  • Miscellaneous operators: void, delete, new
