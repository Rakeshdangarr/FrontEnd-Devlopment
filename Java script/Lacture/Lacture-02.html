types of Variables

There are three ways to declare a variable in JavaScript:

  1. var:This is the oldest way to declare a variable. It is still supported, but it is not recommended to use it anymore.
  2. let:This is a newer way to declare a variable. It is block-scoped, which means that it is only accessible within the block in which it is declared.
  3. const:This is a newer way to declare a constant. A constant is a variable whose value cannot be changed.

different data types in JavaScript:

  • Number:This data type represents numeric values. It can be an integer, a floating-point number, or a special value like NaN or Infinity.
    (example: var age = 25;)
  • String:This data type represents text. It can be a single character, a word, or a sentence.
    (example: var name = "John Doe";)
  • Boolean:This data type represents true or false values. It is often used to control the flow of a program.
    (example: var isLoggedIn = true;)
  • Array:This data type stores a collection of values. The values can be of any data type, including numbers, strings, booleans, and other arrays.
    (example : var numbers = [1, 2, 3, 4, 5];)
  • Object:This data type stores a collection of key-value pairs. The keys can be strings or numbers, and the values can be of any data type.
    (example:var person = { name: "John Doe", age: 25 };)
  • Null:This data type represents the absence of a value. It is often used to indicate that a variable has not yet been initialized.
    (example: var emptyVariable = null;)
  • Undefined:This data type represents a value that has not yet been assigned. It is often used to indicate that a variable has not yet been declared
    (example: var undeclaredVariable;)

what is syntex

Syntax in JavaScript is the set of rules that define a correctly structured JavaScript program. It is similar to the grammar of a language, and it determines how the code should be written and how the browser should interpret it.


Here are some of the basic rules of JavaScript syntax:
  • Statements end with semicolons.: This tells the browser that the statement is complete.
  • Curly braces are used to mark blocks of code.: This can be used to group statements together, or to define a function
  • JavaScript is case-sensitive.: This means that the keywords var, let, and const are different from the keywords VAR, LET, and CONST
  • Comments are used to explain the code.: Comments are ignored by the browser, but they can be helpful for other programmers who are reading the code.

what is keyword

Keywords are reserved words in JavaScript which cannot be used as variable names, function names, object names, or any other identifier. They are used to define the structure of a JavaScript program and to perform specific actions. There are a total of 63 keywords in JavaScript.

Some examples of JavaScript keywords are:

  • var is used to declare a variable.
  • function is used to declare a function.
  • if is used to create an if statement.
  • else is used to create an else statement.
  • for is used to create a for loop.
  • while is used to create a while loop.
  • return is used to return a value from a function.
  • break is used to break out of a loop.
  • continue is used to continue to the next iteration of a loop.

what is comparison operator in javascript

Comparison operators are used to compare two values and return a Boolean value, either true or false. The following table lists the different comparison operators in JavaScript:

Operator |Description with example
  • == |Equal to
    • console.log(10 == 10); // true
    console.log(10 == "10"); // true
  • != |Not equal to
    • console.log(10 != 10); // false
    console.log(10 != "10"); // false
  • === |Strict equal to (compares both value and type)
    • console.log(10 === 10); // true
    console.log(10 === "10"); // false
  • !== |Strict not equal to (compares both value and type)
    • console.log(10 !== 10); // false
    console.log(10 !== "10"); // true
  • > |Greater than
    • console.log(10 > 5); // true
    console.log(10 > 10); // false
  • < |Less than
    • console.log(10 < 5); // false
    console.log(10 < 10); // false
  • >= |Greater than or equal to
    • console.log(10 >= 5); // true
    console.log(10 >= 10); // true
  • <= |Less than or equal to
    • console.log(10 <= 5); // false
    console.log(10 <= 10); // true

Javascript Popup Box

There is three type of popup box in javascript.

  1. Alert box
  2. Confirm Box
  3. Prompt Box

Alert box

An alert box is often used if you want to make sure information comes through to the user. When an alert box pops up, the user will have to click "OK" to proceed.

</script>

Confirm Box

A confirm box is often used if you want the user to verify or accept something. When a confirm box pops up, the user will have to click either "OK" or "Cancel" to proceed. If the user clicks "OK", the box returns true. If the user clicks "Cancel", the box returns false.

Example
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
<title>Example of confirm box</title>
<script type="text/javascript">
function confirmit(){if(confirm("Are You Sure?"))
{alert("You Click On Ok!");}else {alert("You Click On Cancel! ");}}
</script>
</head>
<body>
<input type="button" value="Click Me!" onclick="confirmit()"/>
</body>
</html>

Prompt Box

A prompt box is often used if you want the user to input a value before entering a page. When a prompt box pops up, the user will have to click either "OK" or "Cancel" to proceed after entering an input value. If the user clicks "OK" the box returns the input value. If the user clicks "Cancel" the box returns null.
