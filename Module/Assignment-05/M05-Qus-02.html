Qus: How to embed audio and video in a webpage?
Ans: To embed audio in HTML, we use the < audio > tag. Before HTML5, audio cannot be added to web pages in the Internet Explorer era. To play audio, we used web plugins like Flash. After the release of HTML5, it is possible. This tag supports Chrome, Firefox, Safari, Opera, and Edge in three audio formats – MP3, WAV, OGG. Only Safari browser doesn’t support OGG audio format.

Example

< audio >
< source src="file name" type "audio file type"
< audio >
