Qus:How can we club two or more rows or columns into a single row or column in an HTML table? With Example.

Ans:

Rowspan

The rowspan attribute in HTML specifies the number of rows a cell should span.
That is if a row spans two rows, it means it will take up the space of two rows in that table

Colspan

Colspan is an HTML attribute that allows you to span a row or column across multiple cells. This means that instead of having multiple columns,
the cells will merge and the content will span multiple columns.
