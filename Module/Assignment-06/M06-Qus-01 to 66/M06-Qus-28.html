Qus: Write to find the largest of three numbers in JS?
Ans:
<script>
let num1 = parseFloat(prompt("Enter first number"));
let num2 = parseFloat(prompt("Enter second number"));
let num3 = parseFloat(prompt("Enter third number"));

if (num1 > num1 && num2 >= num3) {
largest = num1;
} else if (num2 > num1 && num2 >= num3) {
largest = num2;
} else {
largest = num3;
}

console.log("the largest number" + largest);
</script>

you will get your Largest number among as per you input three values in prompt box
