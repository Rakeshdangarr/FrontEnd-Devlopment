Qus: How to maintain the CSS specifications?
Ans: Cascading Style Sheets (CSS) can be described as a style sheet language written in a markup language. It is used for defining the look and formatting elements of Web documents such as the layout, colors, and fonts.
  • The CSS specifications are maintained by the World Wide Web Consortium (W3C).
  • Even though every browser supports CSS, there are many inconsistencies in the supported specification version.
  • Some browsers even have their own implementation of the specification and have proprietary (vendor) prefixes.
