Qus: Which property controls the image scroll in the background?
Ans:

The "background-attachment"

property use to background image scrolls with the page, or not scroll.

Example

<style> body { background-image: url("https://images.rawpixel.com/image_800/czNmcy1wcml2YXRlL3Jhd3BpeGVsX2ltYWdlcy93ZWJzaXRlX2NvbnRlbnQvbHIvdjU0NmJhdGNoMy1teW50LTM0LWJhZGdld2F0ZXJjb2
xvcl8xLmpwZw.jpg");
background-repeat: no-repeat;
background-attachment: fixed;
}
</style>
