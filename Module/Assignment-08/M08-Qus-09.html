Qus:

How can one create an alert in Bootstrap ?


Ans:

Bootstrap alert

Bootstrap Alerts offer a simple method for crafting predefined messages, enhancing their appeal to users. They streamline message display, adding style for increased engagement and improved user experience by presenting important information effortlessly and appealingly.


Syntax for alert

<div class="alert alert-success" role="alert"></div>

  • use the .alert class along with the contextual classes that are used to display the alert message in the application.
  • The alert classes like .alert-success, .alert-info, .alert-warning, .alert-danger, .alert-primary, .alert-secondary, .alert-light, and .alert-dark are been used to represent the alert message with different behavior.
  • generate the dynamic alert message when the user clicks on the button.
