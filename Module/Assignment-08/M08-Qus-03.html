Qus:

What are the default Bootstrap text settings?


Ans:
The default text settings for Bootstrap 4 and Bootstrap 5 are:
  • Font size: 16px
  • Line height: 1.5
  • Margin top: 0
  • Margin bottom: 1rem (16px)
  • Font family: "Helvetica Neue", Helvetica, Arial, sans-serif
  • Bootstrap uses the browser's default root font size for a more accessible type scale. You can use the following attributes to set typographic base styles for the <body>: $font-family-base, $font-size-base, and $line-height-base.
  • You can also use the following classes to customize headings: .h1 through .h6, .text-muted, and .display-1 through .display-4
  • To enable responsive font sizes, you can change the $enable-responsive-font-sizes Sass variable to true and recompile Bootstrap.
