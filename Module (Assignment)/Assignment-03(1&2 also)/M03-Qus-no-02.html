Qus: What are tags and attributes in HTML ?
Ans:

Tags

There are two types of tags:
  • Pair tag(Open tag,)

    • Pair tags have range and must have to close after input containt.

  • Non-pair tag(Close tag, Void tag, self tag)

    • Non-pair tags are open & close at the same time.


Attributes

Attributes are instructions that clarify or modify an element. if we say in other way that attributes are the additional things to describe tags.
