Qus: How to insert a picture into a background image of a web page? With Example.

Ans:

Using background attribute We can use the background attribute in the body tag to set an image as the background of the webpage.
We will need to specify the URL or the location of the image which we want to set to the background attribute of the body tag.

Example
< body background="https://img.freepik.com/free-vector/hand-painted-watercolor-pastel-sky-background_23-2148902771.jpg">
