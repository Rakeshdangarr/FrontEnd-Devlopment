Qus: Print the length of the string on the browser console using console.log()?
Ans:
<script>
const str =
"Lorem ipsum dolor sit amet, consectetur adipisicing elit. Quibusdam, officiis! Answer:";

console.log(`${str} ${str.length}`);
</script >
output
Lorem ipsum dolor sit amet, consectetur adipisicing elit. Quibusdam, officiis! Answer: 86
