Qus: Write to find minimum number among 3 numbers using ternary operator in JS?
Ans:
<script>
let input1 = parseFloat(prompt("Enter first number"));
let input2 = parseFloat(prompt("Enter Secon number"));
let input3 = parseFloat(prompt("Enter third number"));
let largestNumber = Math.max(input1, input2, input3);

let smallestNumber = Math.min(input1, input2, input3);

console.log("the minimum number is " + smallestNumber);
</script>

you will get your Minimum number among as per you input three values in prompt box bellow
