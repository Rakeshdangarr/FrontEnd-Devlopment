Qus: What are the media types allowed by CSS?
Ans:CSS allows many media types, including:
  1. Speech
  2. Audio
  3. Visual and tactile media
  4. Paged or continuous media
  5. Bitmap or grid media
  6. Interactive media
  • The most common media types are screen and print. The default state of a media query is "all".
  • CSS media queries allow a website's layout to respond to different screen sizes. This creates a responsive web design, where the view of a web page differs from system to system based on screen or media types.
  • CSS3 media queries allow users to apply CSS rules based on the media type and media features. This makes it easy to apply different CSS rules based on the requesting device.
